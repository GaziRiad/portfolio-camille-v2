Delivery of key screens

ANNA Money offers a SaaS product for busines owners. I led experience design for the dashboard, settings, payroll and tax filing. When a UI designer was not available I designed screens and components too.

Dashboard

For the dashboard I defined an experience that highlights important tasks to users in a timeline format. A metaphorical example that inspired this was the Tesla dashboard (which shows how the engine is running and where you’re headed).

Settings

I wireframed solutions for everything the settings section required and I created UI elements when a designer was not available.

Payroll

Sometimes, product design means having the right idea for the solution, and inspiring a team to release it. For payroll, I defined unique experiences such as an interactive payslip with real-time tax calculations, and smooth, step-by-step onboarding. I also designed screens to monetise the solution.

Customer Journeys Mockups Prototypes User Testing

Tax filing, made easy

ANNA Money's product goals have been to drive engagement and generate subscriptions by delivering easy to use tools for tax filing. I used my UX skills to design simple experiences for complex tasks.

VAT Return Filing

ANNA has clever Machine Learning technology that auto-classifies bank transactions and allocates them to the right section of a tax return. I wireframed detailed solutions to create an experience that makes VAT filing easy for business owners.

Wireframes Prototypes

Tax calculator

Here I grasped complex tax details and transformed them into a user-friendly interface, one that enables business owners to calculate their tax bill. I also recruited interview candidates and tested prototypes early on to gather user feedback before iterating.

Wireframes User Testing Visual Design

Growth experiments

Working in quick cycles, I assisted with idea gen and I rapidly mocked up hi-fi screens so we could test experiments for subscription growth.

UI Design

I mostly conducted UX for this project but I also stepped in to create detailed UI components and visual imagery when required.

Card payments

ANNA has many flows with subscription paywalls. Here I created detailed UI components that help a user to “check out” and complete purchases on repeat using saved card details.

User Journeys

I created a set of user journeys for the payroll project. This helped the wider team understand and agree upon a good experience that we would create together.

Low-fidelity mockups

For the ANNA Money web platform I created a number of wireframes, so that we could easily discuss scope as a team, identify design patterns and obtain approval from stakeholders.

User testing

I created interactive Figma prototypes and conducted a number of tests. After recruiting interview candidates from our internal audience, I asked open-ended questions and observed behaviour while users completed set tasks. I collated interview feedback in a 'rainbow' format in order to share with the team and highlight areas for improvement.

Results

The impact of my efforts included user adoption of new features and customer acquisition onto paying subscriptions.
